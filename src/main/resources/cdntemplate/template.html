1 Typography & Elements

Section Headings

Section headings (h2) use 20px font, weight 800, with a green 3px bottom border. The yellow circle number (.num) is optional. Subsection headings (h3) use a green 4px left border.

Detail headings (h4) — weight 700, muted color

Inline Text

Paragraph with bold, italic, underline, strikethrough, highlighted, inline code, Ctrl+K, HTML, sup and sub. List markers receive the accent color via li::marker.

Highlight & Blockquote

Blockquote with green left border and lightly tinted background. Ideal for technical notes, warnings, or critical path callouts.

Accordion

Item 1 — General information

Expandable content via native <details>. Zero JavaScript required. CSS adds the animated green chevron.

Item 2 — Technical details

PicoCSS styles the <summary> element. The CDNWebTemplate CSS adds consistent borders, typography, and animation.

Item 3 — Open by default

The open attribute defines the initially expanded state. Useful for the first item or the most relevant content.

Card (Article)

Card Title

Content via semantic <article>. Border, padding, and background color adapt automatically to light or dark theme.

Footer with metadata, date, or secondary actions

Dialog / Modal

Action confirmation

Native modal via <dialog> controlled by Alpine.js $refs. Backdrop and styling are applied via CSS — no extra dependencies.
